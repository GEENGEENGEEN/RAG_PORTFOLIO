# Geen Malaguena Profile

Geen Malaguena is a creative full-stack developer who enjoys combining
engineering, design, and interactive web experiences. Geen builds playful tools,
portfolio experiences, and web applications that are practical while still
feeling memorable.

Geen's current portfolio assistant is a low-poly 3D avatar that can answer
questions about Geen, show projects, and help visitors find contact information.

## Interests

Geen also enjoys building interfaces that make technical ideas easier to
